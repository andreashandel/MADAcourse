Course Resources

Andreas Handel

2019-12-04

This page lists resources that are used or mentioned in this course. For a more extensive list of data science resources, see the General Resources page.

All materials described below are (should be) freely available online. If you can’t get to them, let me know. Note that a lot of the listed resources are dynamic and ever changing. That means occasionally links might not work, sites go offline, chapters in online books get re-arranged, etc. If you notice anything that’s not quite right, please let me know.

General help

  • The main place to get any course specific help are the discussion boards on eLC. Use them widely to ask questions, to answer others’ questions, to post links to interesting resources, etc.
  • Most questions you will have are likely not course specific, but will have to do with R/Github/RMarkdown/etc. For that, Google will be your best friend. Most of the time, someone had the same problem/question you do and someone else answered it. The only tricky thing is finding that post. Even after years of doing this, I probably google how to do something in R every day 😃.
  • When you search online for help, quite often you land on some StackExchange site. These are a collection of widely-used online question and answer sites covering all kinds of topics (including R, Github, Data Analysis, etc.). The majority of the time, you will find an answer, or at least something that gets you closer, on those sites.
  • The Rstudio community is another good place to ask questions. It is not as widely used as StackExchange, but it seems to be more newcomer friendly. I haven’t used it much.
  • Another highly recommended resource is the R for data science community. They have a Slack workspace which you can join for free and ask questions about R, Data Science and related topics. They are very newbie-friendly. I have not used them much but they seem to respond fairly quickly and are helpful.
  • Here is another resource, a list of individuals who volunteered to answer R and/or Data Science questions mainly through Twitter.

Github Resources

  • A great resource to learn Git/Github with R is happygitwitR (HGR) by Jenny Bryant.
  • I recommend using a graphical client to do (almost) all your Github work (push, pull, etc.). I use GitKraken which has a fully functional free version. As a student, you can also get a free 1 year subscription to the Pro version (see the GitHub course document for details). Other graphical Git/GitHub clients exist, see e.g. Jenny’s book.
  • For classwork, projects and other materials that we might not want to share publicly, we will use a GitHub organization specifically created for the class. Instructions on how to join the organization and use it are provided in the course documents.
  • If you have questions for a specific R package (or other project) which is developed on GitHub, you can ask questions in the GitHub Issues section. We’ll discuss/practice that as part of the course.

Books

  • Introduction to Data Science (IDS) is the book I’ll refer to a good bit in the 1st part of the course.
  • R for Data Science (R4DS) is another resource that will come up frequently. It is a very gentle and good introduction to data science in R. I’ll be refering to it frequently.
  • The Art of Data Science (ADS) has some good big-picture, introductory level chapters on different aspects of a data analysis. It is a pay what you want book with a minimum price of zero, which means you can get it for free (note that the book + video bundle has a non-zero minimum price).
  • An Introduction to Statistical Learning (ISL) is a good introduction to the statistical and machine learning (i.e. model fitting) part of the data analysis workflow. I’ll refer to it frequently in the second part of the course.
  • Hands-on Machine Learning with R covers somewhat similar material as ISL, but with different emphasis and a different approach. I refer to it in several of the later course modules.
  • Feature Engineering and Selection (FES) focuses on a specific aspect of the data analysis workflow and is a good resource for those topics.

Hands-on/interactive resources

  • Online Tutorials (primers) by Rstudio (RP) are a great source for introductory, interactive tutorials covering basics of R coding and data analysis. We’ll use several of those primers in the class.
